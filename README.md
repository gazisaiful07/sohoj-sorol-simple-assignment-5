What is the difference between var, let, and const?
ans:
>> var is the older way of declaring variables. It is function-scoped and can be re-declared and updated.
>> let is block-scoped. Its value can be updated, but it cannot be re-declared in the same scope.
>> const is also block-scoped, but its value cannot be updated or re-assigned after it is declared.

 What is the spread operator (...)?
 ans:
>> The spread operator (...) in JavaScript is used to expand or copy elements of an array or object into another array or object. It helps to combine arrays, copy arrays, or copy objects easily.

What is the difference between map(), filter(), and forEach()?
ans:
>> map(), filter(), and forEach() are  the array methods used to work with array elements in JS.
deference between them are below---
>> map() is used to create a new array by modifying each element of the original array.
>> filter() is used to create a new array with elements that match a condition from the old array.
>> forEach() is used to loop through an array, but it does not return a new array.

What is an arrow function?

What are template literals?
ans:
>> Template literals are a way to create strings in JavaScript using backticks (` `) instead of quotes. They allow us to insert variables or expressions inside a string easily using ${}
